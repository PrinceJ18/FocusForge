# Project Rules and Guidelines

This file defines workspace-specific style guidelines, behavioral constraints, and general instructions for AI coding agents working on **FocusForge Ad**.

## 🛠️ Project Stack & Environment
- **Frontend Framework**: Vite + React + TypeScript (if applicable)
- **Styling**: TailwindCSS & Vanilla CSS
- **Database/Backend**: Supabase

## 📝 Coding Guidelines
- Follow clean code practices and write self-documenting code.
- Ensure TypeScript types are strictly defined where appropriate (avoid `any` where possible).
- Preserve existing comments and docstrings unless explicitly instructed to change or refactor them.

## 🧪 Testing and Verification
- Verify all changes compile and build successfully.
- Write tests or run verification steps where possible before completing a task.
